compact-classic-gnome-theme: compact, classic GNOME theme
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
compact-classic-gnome-theme is a compact, classic theme for various versions
of GNOME and GTK. It looks similar to how GNOME 2 looked in 2010: no round
edges, no unnecessary whitespace, no animations, no shadows. It builds on
existing standard GNOME themes, and it modifies mostly window decorations
(titles, buttons, borders) and GNOME Panel.

Themes implemented:

* compact_classic_gnome_flashback_gtk3.css : Compact classic theme
  implementation for GNOME Flashback and Metacity 3.24--3.28--3.36-- (works
  with Debian 9 on 2018-03-05, Debian 10 and Ubuntu 18.04) and GTK+
  3.22--3.24--. See installation instructions within the file.
